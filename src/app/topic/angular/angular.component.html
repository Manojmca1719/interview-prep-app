What is Angular?

  • Angular is a component-based Framework for building scalable web applications. built on TypeScript.
  • A collection of well-integrated libraries that covers a wide variety of features, including forms, routing, client-server communication, and more
  • Developed by Google

What is SSR means in Angular?

  • SSR means Server Side Rendering.
  • Normal Angular application executes in the browser.
  • But in Angular Universal, executes on the server side, generating static application pages then later get bootstrapped on the client side.
  • Therefore giving users a chance to view the application layout before it becomes fully interactive.

Which command used for generate Angular Universal?

ng add @nguniversal/express-engine

How to run Angular Universal?

npm run dev:ssr
